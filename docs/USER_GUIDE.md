# User Guide

## Dashboard

Use the Dashboard to inspect overall app state, pause or resume automation, and
see the current operational summary.

## Machine Specs

Use Machine Specs to load the live hardware probe or switch to fixtures for
Apple Silicon, Intel Mac, and Windows GTX 1060 test profiles. Specs can be
exported as JSON, CSV, or Markdown.

## Model Fit Map

Use Model Fit Map to compare catalog models against the active hardware. Labels
range from Smooth to Avoid.

## Providers

Use Providers to inspect Ollama, LM Studio, custom OpenAI-compatible, MLX-LM,
and llama.cpp adapters. Test chat is intended for local verification.

## Models

Use Models for dry-run runtime and model setup plans. Stage 14 does not download
real model weights.

## Router

Use Router to choose Auto, Manual, Forced, Local only, Remote preferred, Remote
only, or Paused routing. The test prompt panel shows which provider/model would
handle a request.

## Remote PCs

Use Remote PCs to manage Windows broker mode and Mac remote client pairing.
Remote pairing supports Bonjour/mDNS discovery and manual broker addresses.

## Updates

Use Updates to check fixture metadata sources, review compatibility labels,
ignore updates, remind later, or run dry-run install actions.

## Settings and Logs

Use Settings for notification and background behavior. Use Logs to inspect pause,
notification, install, update, provider, router, and remote events.
